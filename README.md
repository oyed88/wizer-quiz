Wizer Quiz is a web-based quiz application built with React and Tailwind CSS. It fetches multiple-choice questions from the Open Trivia Database API and allows users to answer questions and view their final score.

Features
Multiple-choice quiz questions
Real-time score tracking
Responsive user interface

Tech Stack
React (Vite)
Tailwind CSS
JavaScript

Getting Started
git clone https://github.com/your-username/wizer-quiz.git
cd wizer-quiz
npm install
npm run dev
